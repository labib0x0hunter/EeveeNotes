### ASM - x86 Assembly
---
8086 microprocessor
# Mac (M1) Setup
- Install DosBox : https://www.dosbox-staging.org/ 
```
mkdir ~/dos/asm
```
- Mount folder as C drive
```
mount c ~/dos/asm
c:
```
- Get jwasm.exe and link.exe (Microsoft Macro Assembler)
```
jwasm -mz hello.asm
hello
```
